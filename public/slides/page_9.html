JavaScript vs TypeScript

TypeScript = JavaScript + Types

JavaScript

  • Dynamic typing
  • No type checking
  • Errors found at runtime
  • Flexible but risky

JavaScript Example: 

function calculatePrice(price, quantity) {
  return price * quantity;
}

calculatePrice("100", 2);  // "100100" ‚ùå Bug!

JavaScript won't warn you about these mistakes!

TypeScript

  • Static typing
  • Type checking at compile time
  • Errors found before running
  • Safe and predictable

TypeScript Example: 

function calculatePrice(price: number, quantity: number): number {
  return price * quantity;
}

calculatePrice("100", 2);  // ‚ùå Error: string not assignable

TypeScript helps you catch errors early and write more reliable code
