Frontend vs Backend Architecture

Frontend (Client-Side)

  • What users see and interact with
  • HTML/CSS/JavaScript
  • Runs in browser

Examples:

Buttons Forms Animations

Backend (Server-Side)

  • What happens behind the scenes
  • Business logic & data processing
  • Runs on server

Examples:

Save data Authenticate users
