Creating Simple API Endpoints

Hello World Endpoint

// GET /api/hello
import express from 'express' ;

const router = express.Router();

router .get( '/' , (req, res) => {
res.json({
message: 'Hello from Backend!' ,
timestamp: new Date()
});
});

URL Parameters

// GET /api/hello/:name
router .get( '/:name' , (req, res) => {
const { name } = req.params; // Get from URL

res.json({
message: `Hello, ${name}!` ,
timestamp: new Date()
});
});

POST Request

// POST /api/hello
router .post( '/' , (req, res) => {
const { name, age } = req.body; // Get from request body

// Validation
if (!name || !age) {
res.status(400).json({ error : 'Name and age are required' });
return ;
}

res.json({
message: `Hello, ${name}! You are ${age} years old.` ,
timestamp: new Date()
});
});

HTTP Request Parts

Method
GET, POST, PUT, DELETE
URL
/api/users, /api/products/:id
Headers
Content-Type, Authorization
Body
JSON data sent with request

Test with: curl -X GET http://localhost:5000/api/hello
