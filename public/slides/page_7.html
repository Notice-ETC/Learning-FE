HTTP Request and Response Structure

HTTP Request

POST /api/users HTTP/1.1
Host: localhost:5000
Content-Type: application/json

Method

Verb indicating operation (GET, POST, PUT, DELETE)

URL

Resource location (/api/users)

Headers

Metadata (Content-Type, Authorization)

Body

Request data (JSON)

HTTP Response

HTTP/1.1 201 Created
Content-Type: application/json

Status Code

Three-digit code indicating result (200, 201, 400, 404, 500)

Headers

Metadata (Content-Type, Cache-Control)

Common Status Codes:

200 OK
201 Created
400 Bad Request
404 Not Found
500 Server Error

HTTP is the foundation of data communication on the web
