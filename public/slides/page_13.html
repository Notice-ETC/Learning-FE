Error Handling and API Testing

Error Handling

Use try-catch blocks for proper error management: 

router.get('/:id', async (req: Request, res: Response) => {
  try {
    const { id } = req.params;
    const user = await User.findOne({ id });
    
    if (!user) {
      return res.status(404).json({ error: 'User not found' });
    }
    
    res.status(200).json({ data: user });
  } catch (error) {
    res.status(500).json({
      error: 'Internal server error',
      details: error.message
    });
  }
});

Response Methods

  • res.json() - Send JSON response (most common)
  • res.send() - Send plain text
  • res.sendStatus() - Send status code only
  • res.redirect() - Redirect to URL
