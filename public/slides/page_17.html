Testing with cURL

Create Product (POST)

curl -X POST http://localhost:5000/api/product \ -H "Content-Type: application/json" \ -d '{"id":"P001","name":"Laptop","price":25000,"stock":10,"category":"Electronics"}'

Get All Products (GET)

curl http://localhost:5000/api/product

Update & Delete

curl -X PUT http://localhost:5000/api/product/P001 \ -H "Content-Type: application/json" \ -d '{"price":22000,"stock":8}'
curl -X DELETE http://localhost:5000/api/product/P001
